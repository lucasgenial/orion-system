# App Painel ORION

O app-painel e a frente analitica do ecossistema ORION. Nesta etapa, o diretorio passa a ter uma base inicial de prototipo para orientar identidade, densidade visual e organizacao de informacoes executivas.

## Objetivo

O painel foi pensado para consolidar indicadores vindos do app-operacional e transformar o dado transacional em leitura gerencial, acompanhamento tatico e visao executiva.

## Entregue nesta fase

- landing inicial do painel em HTML estatico;
- identidade visual propria com linguagem de centro de comando;
- blocos de KPI, tendencia, mapa operacional, alertas e agenda executiva;
- interacao basica por filtros de periodo e foco;
- base pronta para futura migracao para framework frontend ou dashboard server-side.

## Como visualizar

Abra o arquivo index.html no navegador ou sirva a pasta por um servidor estatico simples.

## Proximos passos sugeridos

- definir a stack definitiva do painel;
- conectar indicadores reais vindos do app-operacional;
- desenhar paginas de drill-down por unidade, territorio, produtividade e risco;
- criar biblioteca compartilhada de tokens visuais do ecossistema ORION.
